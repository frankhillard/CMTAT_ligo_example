# CMTAT token implementation example
Example of a swiss-law compliant (and TZIP-12 compliant) token using the Ligo registry (@ligo/cmtat library)

## Compile contract
```
make install
make compile
```

## deploy contract
Taquito requires node (version 18+)

```
nvm install v18.19.0
nvm use v18.19.0
make deploy
```
